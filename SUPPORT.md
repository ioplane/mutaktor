# Support

## Getting Help

| Channel | Use for |
|---------|---------|
| [GitHub Issues](https://github.com/ioplane/mutaktor/issues) | Bug reports, feature requests |
| [GitHub Discussions](https://github.com/ioplane/mutaktor/discussions) | Questions, ideas, show & tell |

## Bug Reports

When filing a bug report, include:

- **mutaktor version** (from `gradle.properties` or plugin portal)
- **Gradle version** (`./gradlew --version`)
- **JDK version and vendor** (`java -version`)
- **PIT version** (if overridden from default)
- **OS and architecture**
- **Full stack trace** (run with `--stacktrace`)
- **Relevant `build.gradle.kts`** sections

## Feature Requests

Describe the problem you're trying to solve, not just the solution you want. Include examples of how you'd use the feature in your build.
