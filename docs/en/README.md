![License](https://img.shields.io/badge/License-Apache_2.0-blue?style=flat-square)

> Also available in: [Russian](../ru/README.md)

**mutaktor** is a Kotlin-first Gradle plugin for [PIT](https://pitest.org/) mutation testing. It adds git-aware scoping, Kotlin junk-mutation filtering, standardized reporting (SARIF, mutation-testing-elements JSON), and GitHub CI/CD integration on top of PIT's proven mutation engine.

## Module Overview

| Module | Plugin ID | Purpose |
|--------|-----------|---------|
| `mutaktor-gradle-plugin` | `io.github.dantte-lp.mutaktor` | Applied to consumer projects; registers `mutate` task |
| `mutaktor-gradle-plugin` | `io.github.dantte-lp.mutaktor.aggregate` | Applied to root project; registers `mutateAggregate` task |
| `mutaktor-pitest-filter` | _(PIT SPI)_ | Loaded by PIT at runtime; filters Kotlin compiler-generated mutations |
| `build-logic` | _(internal)_ | Shared Kotlin + JVM toolchain convention plugin |
