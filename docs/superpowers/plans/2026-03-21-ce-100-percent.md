# CE 100% — Wire All Features End-to-End

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Wire all existing converters, quality gate, ratchet, and reporters into the MutaktorTask execution pipeline so that `./gradlew mutate` produces JSON, SARIF, evaluates quality gate, runs ratchet, and reports to GitHub — all in one command.

**Architecture:** Add `doLast {}` post-processing to MutaktorTask that reads PIT's `mutations.xml` output and runs converters/evaluators. Each feature is gated by extension properties (off by default for heavy features, on by default for reports).

**Tech Stack:** Kotlin 2.3, Gradle 9.4.1, PIT 1.23.0

---

## Current State

MutaktorTask.exec() runs PIT → produces `mutations.xml` + HTML in `reportDir`.
After that: **nothing**. Converters, quality gate, ratchet, GitHub reporter all exist as standalone classes but are never called.

## Target State

```
./gradlew mutate
  ├── PIT execution (existing) → mutations.xml + HTML
  ├── Post-processing (NEW):
  │   ├── MutationElementsConverter → mutations.json (always)
  │   ├── SarifConverter → mutations.sarif.json (if sarif=true)
  │   ├── QualityGate → pass/fail (if threshold set)
  │   ├── MutationRatchet → baseline check (if ratchet enabled)
  │   └── GithubChecksReporter → PR annotations (if GITHUB_TOKEN present)
  └── BUILD SUCCESS or FAILURE
```

- [ ] Override exec() to add post-processing after super.exec():

```kotlin
override fun exec() {
    mainClass.set(PIT_MAIN_CLASS)
    classpath = launchClasspath
    if (useClasspathFile.getOrElse(false)) writeClasspathFile()
    args(buildPitArguments())
    logger.lifecycle("Mutaktor: launching PIT {} with {} argument(s)", pitVersion.getOrElse("(unknown)"), args?.size ?: 0)

    super.exec()

    // Post-processing: convert reports + evaluate gates
    postProcess()
}

private fun postProcess() {
    val mutationsXml = reportDir.get().asFile.resolve("mutations.xml")
    if (!mutationsXml.exists()) {
        logger.warn("Mutaktor: mutations.xml not found — skipping post-processing")
        return
    }

    val sourceRoot = sourceDirs.files.firstOrNull() ?: reportDir.get().asFile

    // 1. JSON report (mutation-testing-elements)
    if (jsonReport.getOrElse(true)) {
        val json = MutationElementsConverter.convert(mutationsXml, sourceRoot)
        val jsonFile = reportDir.get().asFile.resolve("mutations.json")
        jsonFile.writeText(json)
        logger.lifecycle("Mutaktor: wrote mutation-testing-elements JSON → {}", jsonFile)
    }

    // 2. SARIF report
    if (sarifReport.getOrElse(false)) {
        val sarif = SarifConverter.convert(mutationsXml, pitVersion.getOrElse("unknown"))
        val sarifFile = reportDir.get().asFile.resolve("mutations.sarif.json")
        sarifFile.writeText(sarif)
        logger.lifecycle("Mutaktor: wrote SARIF 2.1.0 → {}", sarifFile)
    }

    // 3. Quality gate
    if (mutationScoreThreshold.isPresent) {
        val result = QualityGate.evaluate(mutationsXml, mutationScoreThreshold.get())
        logger.lifecycle("Mutaktor: mutation score {}% (threshold {}%) — {}",
            result.mutationScore, result.threshold, if (result.passed) "PASSED" else "FAILED")
        if (!result.passed) {
            throw GradleException("Mutaktor quality gate FAILED: score ${result.mutationScore}% < threshold ${result.threshold}%")
        }
    }

    // 4. Ratchet
    if (ratchetEnabled.getOrElse(false)) {
        val current = MutationRatchet.computeScores(mutationsXml)
        val baselineFile = ratchetBaseline.get().asFile
        val baseline = if (baselineFile.exists()) RatchetBaseline.load(baselineFile) else emptyMap()
        val ratchetResult = MutationRatchet.evaluate(current, baseline)

        if (ratchetAutoUpdate.getOrElse(true) && ratchetResult.passed) {
            RatchetBaseline.save(baselineFile, current)
            logger.lifecycle("Mutaktor: ratchet baseline updated → {}", baselineFile)
        }

        if (!ratchetResult.passed) {
            val regs = ratchetResult.regressions.joinToString("\n") {
                "  ${it.packageName}: ${it.previousScore}% → ${it.currentScore}%"
            }
            throw GradleException("Mutaktor ratchet FAILED — score regressions:\n$regs")
        }
        logger.lifecycle("Mutaktor: ratchet PASSED ({} packages, {} improved)",
            current.size, ratchetResult.improvements.size)
    }

    // 5. GitHub Checks (if token available)
    val githubToken = System.getenv("GITHUB_TOKEN")
    val githubRepo = System.getenv("GITHUB_REPOSITORY")
    val githubSha = System.getenv("GITHUB_SHA")
    if (githubToken != null && githubRepo != null && githubSha != null) {
        val gate = QualityGate.evaluate(mutationsXml, mutationScoreThreshold.getOrElse(0))
        val survivors = QualityGate.survivedMutants(mutationsXml)
        GithubChecksReporter.report(githubToken, githubRepo, githubSha,
            survivors, gate.mutationScore, mutationScoreThreshold.getOrElse(0))
        logger.lifecycle("Mutaktor: GitHub Check Run created with {} annotations", survivors.size)
    }
}
```
